⚡ Pusher Motor Direction
Before prop install: verify motor spins correct pusher direction. Swap any two bullet phase wires to reverse. Confirm at low throttle — prop should push air rearward. Do this with prop removed.
⚠ Bullet Connector Sizing
AOS Supernova 2807 typically uses 3.5mm bullets. All three phase connections must be fully seated and heat-shrink insulated. Phase order does not matter for direction — swap any two to reverse spin.

// 05 — STABILIZATION

AS3X + SAFE Setup

AR631T+ Orientation

Mount flat and level inside fuselage, aligned with roll and pitch axes. Use the Spektrum Mobile App to define receiver orientation if not in default position.

✓ Binding Procedure
1. TX off. Insert bind plug into BIND port on AR631T+. Power receiver — LED flashes rapidly. Remove bind plug. Put TX in bind mode. LED goes solid = bound. Power cycle both to confirm.

AS3X Starting Gains

ROLL GAIN25%
PITCH GAIN30%
YAW GAIN20%
⚠ Oscillation = Reduce Gain
If rapid oscillation occurs on any axis, reduce that axis gain 20% at a time. Reverse-swept flying wings can be roll-twitchy. Stay conservative on maiden flights.

ℹ Yaw Authority Note
No rudder surface — SAFE yaw correction is limited. Expect some yaw slip in tight turns at low speed. Normal for a flying wing. AS3X yaw gain helps but cannot fully correct.
